For every substantial interaction, create a session file. Run:

```
cogforge wiki session-new --summary "<user intent>" --domain "<domain>" --files-changed "path1,path2" --next-steps "step1,step2"
```

The session file should capture:
- user intent
- sources consulted
- actions taken
- files changed
- decisions made
- unresolved issues
- next steps
- durable knowledge persisted
